# Hello World

**seunan.dev**에 오신 것을 환영합니다.

이 블로그는 터미널 인터페이스로 동작합니다. 명령어를 입력해서 글을 탐색해보세요.

## 사용 가능한 명령어

- `ls` — 글 목록 보기
- `cat <slug>` — 글 읽기
- `grep <keyword>` — 검색
- `ask "질문"` — AI에게 질문

## 코드 예시

```typescript
const blog = {
  name: "seunan.dev",
  theme: "catppuccin-mocha",
  engine: "terminal",
};

console.log(`Welcome to ${blog.name}`);
```

> 터미널은 개발자의 가장 친숙한 도구입니다.
